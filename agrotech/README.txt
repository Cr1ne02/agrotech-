=====================================
   АГРОТЕЧ - Аренда сельскохозяйственной техники
=====================================

Веб-приложение для компании по аренде сельскохозяйственной техники.
Полнофункциональный сайт с каталогом, бронированием, отзывами,
личным кабинетом и админ-панелью.


-------------------------------------
   ЗАПУСК ПРОЕКТА
-------------------------------------

Требования: Node.js версии 16 или выше (https://nodejs.org)

1. Откройте терминал (cmd или PowerShell)
2. Перейдите в папку проекта:
   cd C:\Users\Say2night\Desktop\agrotech

3. Установите зависимости (один раз):
   npm install

4. Запустите сервер:
   npm start

5. Откройте в браузере:
   http://localhost:3000

Для остановки сервера нажмите Ctrl+C в терминале.

-------------------------------------
   БАЗА ДАННЫХ
-------------------------------------

Тип: SQLite (через библиотеку sql.js)
Файл: agrotech.db (в корне проекта)

База данных создается АВТОМАТИЧЕСКИ при первом запуске сервера.
Устанавливать отдельно ничего не нужно.

Таблицы в базе данных:

1. users - Пользователи
   - id, name, email, phone, password, role, created_at
   - Роли: user (обычный), admin (администратор)
   - Пароли хранятся в зашифрованном виде (bcrypt)

2. products - Каталог техники
   - id, name, description, price_per_day, category, image, available, created_at
   - Категории: Тракторы, Комбайны, Сеялки, Плуги, Опрыскиватели, Косилки, Культиваторы

3. bookings - Бронирования
   - id, user_id, product_id, start_date, end_date, total_price, status, created_at
   - Статусы: pending (ожидает), confirmed (подтверждено), completed (завершено), cancelled (отменено)

4. reviews - Отзывы
   - id, user_id, product_id, rating (1-5), comment, created_at

5. cart - Корзина
   - id, user_id, product_id, start_date, end_date

При первом запуске автоматически создаются:
- Аккаунт администратора
- 8 единиц техники в каталоге

Сброс базы данных:
Удалите файл agrotech.db и перезапустите сервер.
Все данные будут созданы заново.

-------------------------------------
   РЕШЕНИЕ ПРОБЛЕМ
-------------------------------------

Сервер не запускается:
- Убедитесь что Node.js установлен: node --version
- Убедитесь что зависимости установлены: npm install

Порт 3000 занят:
- Закройте другие приложения на этом порту
- Или измените PORT в файле server.js

Забыли пароль админа:
- Удалите файл agrotech.db
- Перезапустите сервер (npm start)
- Войдите с данными: admin@agrotech.ru / AgR0T3ch$2026!

Фото не отображаются:
- Проверьте что файлы есть в папке images/
- Имена файлов должны совпадать с указанными в базе данных
